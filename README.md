## Setup

### Install bun

https://bun.sh/

### Install docker

https://docs.docker.com/engine/install/

### Instal pnpm

https://pnpm.io/installation

### Install dependencies

`pnpm install`

### Run database container

`docker-compose up`

### Setup database tables

`pnpm migrate:run`

### Seed database tables

`pnpm seed`

### Inspect database

`pnpm studio`
